# Walkthrough - Update Dokumentasi

Dokumen ini menjelaskan perubahan yang dilakukan pada sesi pembaruan dokumentasi "Jadwal Shalat PWA".

## Ringkasan Perubahan
Membuat direktori `docs/` untuk memusatkan seluruh informasi proyek agar mudah dibaca oleh pengembang maupun pengguna.

## File yang Dibuat

### 1. [README.md](./README.md)
Halaman depan dokumentasi. Berisi apa itu aplikasi ini, fitur kuncinya, dan cara menjalankannya di komputer lokal.

### 2. [TECHNICAL.md](./TECHNICAL.md)
Ditujukan untuk **Developer**. Menjelaskan:
- Bagaimana kode `main.js` bekerja (State management, Fetch API).
- Struktur HTML dan CSS.
- Integrasi API Aladhan dan BigDataCloud.

### 3. [USER_GUIDE.md](./USER_GUIDE.md)
Ditujukan untuk **End-User**. Menjelaskan:
- Cara memberikan izin lokasi.
- Cara mencari kota manual.
- Cara mengubah metode perhitungan shalat (Kemenag, dll).

### 4. [DEPLOYMENT.md](./DEPLOYMENT.md)
Panduan cara meng-online-kan website menggunakan layanan gratis seperti:
- Vercel
- Netlify
- GitHub Pages

## Verifikasi
Semua file di atas telah dibuat dan disimpan di folder `docs/`. Anda dapat membukanya dengan Markdown Reader atau teks editor biasa.
